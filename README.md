## commands

### !나즈나 `캐릭터 이름`

제일 좋아하는 캐릭터의 사진을 출력합니다.

입력하지 않을 시 캐릭명이 `나즈나` 로 검색됩니다.

### !대통령 `이름`

그분들의 사진을 출력합니다.

대통령 기준은 [나](https://github.com/johannblue)는 모름.

### !유저검색 `닉네임`

메이플 gg로부터 데이터를 크롤링하여 해당 캐릭터의 정보를 불러옵니다.

- 사진
- 코디정보
- 레벨
- 인기도
- 무릉도장 최고 기록

### !꺼져, !꼬맹

단순 출력 멘트.
